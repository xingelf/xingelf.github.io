Placing it on the home button makes toggling OFF/ON easier.

iPhone Reading Aloud Challenges

  • Pauses on image pages - Japanese books often have numerous inserted images, unlike English books’ translations, resulting in frequent pauses.
  • Fluency - English books pose no issue, but Japanese books sometimes encounter difficulties in reading aloud.
  • Functionality - Instances where reading aloud fails to start, indicating slight instability.

Quick Start

Preparations

  • If using Suica, sync Wallet to iCloud, delete the app, and reinstall it on the new device, restoring with the registered account.
  • Temporarily disable MAC address restrictions on WiFi, later re-enable with private address registration.
  • Prepare cards if using Apple Wallet, as the 3-digit code is required for migration.
  • Disable automatic startup of VPN software to avoid interruptions during migration and subsequent app installations.

Migration Process

  1. Power on the new device.
  2. Bring it close to the old device.
  3. Capture the screen of the new device.
  4. Connect to WiFi.
  5. Migrate Apple Wallet (input the 3-digit code).
  6. Initiate data transfer from the old device. Depending on data usage, this process took around 10-15 minutes.
  7. The device will restart automatically upon completion of migration.

Automatically Migrated Devices

The following gadgets will be automatically paired:

  • Apple Watch
  • AirPods

Post-Migration Tasks

  1. Connect to WiFi using private address and revert MAC address restrictions.

  2. Uninstall unnecessary default apps:

    • Numbers
    • Keynote
    • Pages
    • Clips
    • iMovie
    • GarageBand

  3. Swap SIM cards.

  4. Migrate Microsoft Authenticator:

    • Begin recovery.
    • Log in with Microsoft account.
    • Permit login notification on the old device (e.g., select a two-digit number).
    • Ensure the same number appears on both old and new devices for accounts with 2FA.

  5. Migrate Google Authenticator:

    • Launch Google Authenticator on the old device.
    • Tap “…” on the top right and select “Export Accounts.”
    • Scan the QR code displayed on the old device with the new one.
    • Choose to delete or keep accounts on the old device. If the same number appears on both devices, it’s safe to delete accounts from the old device.

  6. Restore Suica via the Wallet app. The Suica ID will be changed. After changing devices, errors occurred at automatic gates when using Express on the Shinkansen. If you use the Express app, change the Suica ID linked with it accordingly.

  7. Kindle/Audible: Content needs to be redownloaded. Though it’s unclear if this is intended behavior, synchronization with Apple Watch will also be disabled. It’s a bit inconvenient, but you’ll need to sync again.
