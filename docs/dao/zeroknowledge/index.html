ゼロ知識証明とは

ゼロ知識証明（zero-knowledge proof）とは、ある人が他の人に、 自分の持っている命題（通常、数学的な）が真であることを伝えるのに、 真であること以外の何の知識も伝えることなく証明できるようなやりとりの手法。
