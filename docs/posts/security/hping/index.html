hping/hping3使い方

hping/hping3は、ネットワークホストに様々な種類のパケットを送信し、その応答を分析することができるコマンドラインユーティリティです。

ネットワークスキャン、ファイアウォールテスト、ネットワークテストなど、さまざまなタスクに使用することができます。

hping3 コマンドサンプル 1

  • eth0 インターフェース
  • 10パケット
  • 2.2.2.2で送信元を偽装
  • TTLを100
hping3 -I eth0 -c 10 -a 2.2.2.2 -t 100 [targetIP]

hpingコマンドサンプル 2

  • TCP port 80
  • SYN Packet
hping3 -S -p 80 [targetIP]
