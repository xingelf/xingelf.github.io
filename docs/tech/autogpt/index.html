OpenAIのAPIアカウントを作成する

OpenAIのWebサイトにアクセスし、アカウントを作成します。アカウントを作成すると、APIキーを取得できます。APIキーは、autoGPTを使用するために必要な認証情報です。

https://platform.openai.com/account/api-keys

.env　ファイル編集

 22 ### OPENAI
 23 # OPENAI_API_KEY - OpenAI API Key (Example: my-openai-api-key)
 24 # TEMPERATURE - Sets temperature in OpenAI (Default: 1)
 25 # USE_AZURE - Use Azure OpenAI or not (Default: False)
 26 OPENAI_API_KEY='paste API Key here'
 27 TEMPERATURE=0
 28 USE_AZURE=False
