🛠️ Essential Vim Operations

📂 File Operations

Operation Command
Open file :e
Save file :w
Quit :q
Force quit :q!

🔄 Mode Switching

Operation Command
Enter Insert mode i
Insert at line start I
Insert at line end A
Insert new line below o
Return to Normal mode Ctrl + [ / Esc

💡 Pro Tip
Keyboard layouts vary - train yourself to use Ctrl+[ instead of Esc for smoother workflow!

🚗 Cursor Navigation

Operation Command
Scroll down Ctrl + f
Scroll up Ctrl + b
Jump to first line gg
Jump to last line G

⛓️ Line Joining

Press J (Shift+j) on the line you want to join

🚀 Advanced Vim Techniques

✂️ Cut/Paste Specific Ranges

  1. Move to starting line
  2. Set mark with ms
  3. Move to ending line
  4. Delete marked area with d ' s
  5. Paste with p

📏 Batch Indentation

  1. Shift + v (visual line mode)
  2. Select target lines
  3. Shift + > (indent right)

🌟 Full-file Indentation

  1. Shift + v → Shift + g (select all lines)
  2. >> (indent entire file)

🔍 Interactive Replace

:%s/old_text/new_text/gc

🧩 Plugin Management

🔌 Vundle Plugin Setup

https://github.com/VundleVim/

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

After launching Vim, execute:

:PluginInstall

🐍 Python Development Setup

⚡ JEDI-VIM Installation

Autocompletion powerhouse for Python

git clone --recursive https://github.com/davidhalter/jedi-vim.git ~/.vim/bundle/jedi-vim

Add to your .vimrc:

Plugin 'davidhalter/jedi-vim'
