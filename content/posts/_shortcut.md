# OSX/Macショートカットメモ

## OS全般

### Spotlight Search

```
Command + Space
```

### 画面ロック

```
Control + Command + q
```

---

## 編集

### コピー

```
Command + c
```

### ペースト

```
Command + v
```

### 書式なしペースト　

```
Command + Option + Shift + v
```
orz


### もとに戻す/Undo

```
Command + z
```

### もとに戻すを戻す/Redo

```
Command + Shift + z
```

## 画面キャプチャ

### 矩形画面キャプチャ

```
Command + Shift + 4
```
