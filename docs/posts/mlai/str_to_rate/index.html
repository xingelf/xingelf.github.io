Pythonミニ関数 String/Object型の列にパーセントがある場合、数字に変換

割合の情報がString/Object型の列にパーセントとして与えられた、数字に変換

def str_to_rate(s):
    if pd.isnull(s)==False:
        return float(s.replace('%',''))
    else:
        return s

使い方

col = "ReplyRate"
df[col] = df[col].apply(str_to_rate)
