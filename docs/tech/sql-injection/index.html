4.2 Defense-in-Depth

  • Input Validation: Whitelist allowed characters (e.g., ^[a-zA-Z0-9_]+$).
  • Least Privilege: Restrict DB accounts to SELECT/INSERT; revoke DROP/EXECUTE.
  • WAF Rules: Block patterns like UNION SELECT, ; DROP, and WAITFOR DELAY.

4.3 Detection & Response

  • Behavioral Monitoring: Alert on abnormal query lengths (>500 chars) or rapid UNION attempts.
  • Log Analysis: Centralize logs for anomalies (e.g., 500 errors spiking post-login).
